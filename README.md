# Permission Management System

## Project Description
The Permission Management System enables secure and efficient management of user permissions across various systems and applications. 

## Objectives
- To streamline the permission assignment process.
- Enhance security by minimizing unauthorized access.
- Provide an intuitive user interface for managing permissions.

## Scope
This project focuses on permission management for enterprise applications with extensibility for future integrations.

## Technologies
- Frontend: React.js
- Backend: Node.js
- Database: Firebase
- Authentication: JWT

1. Clone the repository: `git clone https://github.com/sgalindom/permission-management-system.git`

3. Install dependencies: `npm install`
4. Start the application: `npm start`
